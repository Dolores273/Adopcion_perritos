# 🐾 Centro de Adopción de Perritos

Este es un sistema web desarrollado con **Python** y **Flask** para gestionar la adopción de mascotas. El objetivo del proyecto es facilitar la visualización de perritos disponibles y llevar un registro de las solicitudes de adopción.

## 🚀 Funcionalidades
* **Catálogo de Mascotas:** Visualización de perritos con sus detalles.
* **Gestión de Adopciones:** Formulario para solicitar la adopción.
* **Historial:** Registro de las adopciones realizadas.
* **Diseño Responsivo:** Interfaz adaptable para dispositivos móviles y escritorio.

## 🛠️ Tecnologías utilizadas
* **Backend:** Python 3.x y Flask.
* **Base de datos:** SQLAlchemy (SQLite).
* **Frontend:** HTML5, CSS3 y Jinja2.

## 📦 Instalación y Configuración

1.  **Clonar el repositorio:**

    ```bash
    git clone [https://github.com/Dolores273/Adopcion_perritos.git](https://github.com/Dolores273/Adopcion_perritos.git)
    cd Centro_Adopcion
    ```

2.  **Crear y activar el entorno virtual:**
